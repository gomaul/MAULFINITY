# MAULFINITY PROJECT CONTEXT

## Project Name

Maulfinity


## Current Version

Architecture Phase v1.0


## Purpose

Maulfinity is a local-first desktop streaming automation platform.

Main goals:

- TikTok integration
- YouTube integration
- Gift controlled game interaction
- OBS automation
- Overlay creation
- TTS
- Plugin system
